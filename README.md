# vue-2-starter-packs

## Project setup

```
mkdir new-project
cd new-project/
git clone git@github.com:TheSoftwareHouse/vue-2-starter-pack.git .
rm -r .git
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Run unit tests coverage

```
npm run coverage
```

### Run your end-to-end tests

```
npm run test:e2e
```
